# Slack Clone

A full-featured Slack-like team chat app built with **Next.js 15**, **Convex**, **Tailwind CSS**, **TypeScript**, and **pnpm**. Built for interviews: every button and CRUD operation works.

## Features

- **Workspaces** – Create, list, update, delete workspaces
- **Channels** – Create, list, update, delete channels; join and send messages
- **Direct messages (DMs)** – Start DMs with any user, send and delete messages
- **Messages** – Send, list, edit, delete in channels; send and delete in DMs
- **Real-time updates** – Convex keeps all clients in sync
- **Polished UI** – Slack-style sidebar, modals, and Framer Motion animations

## Setup (first time)

### 1. Install dependencies

```bash
pnpm install
```

### 2. Set up Convex (required)

You need a Convex deployment so the app can talk to the backend and so Convex can generate the client API.

```bash
pnpm convex:dev
```

- If prompted, log in or sign up at [convex.dev](https://convex.dev).
- Create a new Convex project (or use an existing one).
- Convex will generate `convex/_generated/*` and add a `.env.local` with `NEXT_PUBLIC_CONVEX_URL`.

Keep this terminal running while developing so Convex stays in sync.

### 3. Run the app

In a **second** terminal:

```bash
cp .env.local.example .env.local
# Edit .env.local and set NEXT_PUBLIC_CONVEX_URL (Convex dev already wrote it if you ran step 2)
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

### 4. Use the app

1. Enter a **name** and **email** and click **Join** (creates your user in Convex).
2. Create a **workspace** from the workspace dropdown → “Create workspace”.
3. Select that workspace, then create a **channel** via the **+** next to “Channels”.
4. Click a channel to open it, then send messages. Use the pencil/trash on your messages to edit/delete.
5. Use **+** next to “Direct messages” to start a DM with another user (open in another browser/incognito to simulate a second user).

## Scripts

| Command | Description |
|--------|-------------|
| `pnpm dev` | Start Next.js dev server |

| `pnpm convex:deploy` | Deploy Convex to production |
